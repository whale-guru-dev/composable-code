# :department_store: Marketplace

Energi NFT Marketplace

## Prerequisites

- Node version 16 (latest)

## Clone the project

```sh
git clone git@git.energi.software:energi/tech/defi/nft/marketplace.git
cd marketplace
```

## Install dependencies

```sh
yarn
```

### Development

```sh
yarn dev
```

### Deployment

```sh
yarn build
yarn start
```
